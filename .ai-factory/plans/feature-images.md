# Добавление картинок в задачи и теорию

## Контекст и цели
В базе у нас уже есть поле `image_url` в модели `Question`. И мы хотим добавить такое же поле в модель `Topic` для теории.  
Нужно:
1. Добавить шаг добавления картинки при создании вопроса в админке.
2. Поддержать отображение этой картинки во всех 4 режимах бота (Спринт, Тренировка, Темы, Работа над ошибками).
3. При необходимости добавить миграцию для поля `image_url` у тем (если его нет).
4. **Важно:** Сохранить работоспособность всех вопросов без картинок (так как большинство вопросов — только текст).
